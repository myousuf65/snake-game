# Snake Game

A simple grid-based movement game built with HTML, CSS, and JavaScript.

## Grid Preview

![Grid](grid.png)

## How to Run

Open `index.html` in a web browser.

## Controls

- Arrow Up: Move up
- Arrow Down: Move down
- Arrow Left: Move left
- Arrow Right: Move right

The active cell (highlighted in red) will move continuously in the chosen direction until a new direction is selected.
